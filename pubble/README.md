# Pubble 🧩

**퍼즐 플랫포머 게임**

Pubble은 웹 기반의 퍼즐 플랫포머 게임입니다. 15x15 그리드에서 스위치를 조작해 벽을 생성/제거하며 목표 지점에 도달하는 것이 목표입니다.

## 🎮 게임 특징

- **15x15 그리드** 기반 게임 보드
- **1칸 점프** 제한으로 정교한 전략 필요
- **스위치 시스템** - 스위치를 밟으면 연결된 벽들이 토글됩니다
- **3개의 기본 레벨** + 무제한 커스텀 레벨
- **레벨 에디터** - 직관적인 드래그 앤 드롭 방식으로 레벨 제작

## 🕹️ 조작법

- **WASD** 또는 **방향키**: 이동
- **스페이스바** 또는 **W**: 점프 (1칸만 가능)
- **S** 또는 **아래 방향키**: 빠르게 하강

## 🛠️ 레벨 에디터

### 기본 도구
- **🧱 벽**: 플랫폼 생성
- **🔴 스위치**: 상호작용 가능한 스위치
- **🏁 목표**: 클리어 지점
- **🗑️ 지우기**: 타일 제거

### 스위치 연결 시스템
1. 스위치 배치 후 자동으로 선택됨
2. "벽 연결 모드" 활성화
3. 연결할 벽들 클릭
4. "연결 완료"로 모드 종료

### 레벨 관리
- **테스트 플레이**: 즉시 레벨 테스트
- **JSON 내보내기/가져오기**: 레벨 공유 및 백업
- **로컬 저장**: 브라우저에 레벨 저장

## 🚀 실행 방법

`index.html` 파일을 브라우저에서 열어 게임을 시작하세요.
